Installing#

To install the Ticketing Integration application from the GitHub repository, complete the following steps:

  1. Contact your Rackspace Account Manager for the Rackspace Ticketing Integration Update Set v1.xml file.
  2. Open ServiceNow.
  3. Navigate to System Update Sets > Retrieved Update Sets.
  4. Click Import Update Set from XML.
  5. Click Choose File and select Rackspace Ticketing Integration.
  6. Click Upload.
  7. After the file is uploaded, click Preview Update Set.
  8. After you preview the file, Click Commit Update Set.
  9. Assign the roles integration_hub_user and x_racus_integratio.integration_hub_user to any user that needs to configure or administer your ServiceNow instance.
