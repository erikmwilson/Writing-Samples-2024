Release Notes
=============

**Rackspace Private Cloud Powered By OpenStack**

*Last updated:* |today|

This document describes new features and known and resolved issues in
Rackspace Private Cloud Powered By OpenStack (RPCO), release |release|
(\ |version|\ ).

RPCO |release| uses open source Ansible playbooks to deploy an
OpenStack |os-release| private cloud. The RPCO repository is available
at https://github.com/rcbops/rpc-openstack. For more information about
the OpenStack-Ansible |os-release| release, see the `Release Notes
<http://docs.openstack.org/releasenotes/openstack-ansible/newton.html>`_
or `OpenStack Configuration Reference
<http://docs.openstack.org/newton/config-reference/>`_.
