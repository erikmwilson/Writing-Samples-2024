Configure a dynamic threshold#

For information about alarms operation (sliding window analysis, static alarms, and dynamic alarms), see AppFormix Alarms Operation in the AppFormix User Guide.

When using a dynamic threshold, you begin by defining an alarm. To configure an alarm, see Alarm Definition in the AppFormix User Guide.

To configure an alarm with a dynamic threshold, you must complete additional fields as described here in the Alarm Definition procedure.
