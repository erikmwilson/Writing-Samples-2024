Optimization and Visualization User Guide#

Rackspace Private Cloud Powered By Openstack

Optimization and Visualization Powered By AppFormix from Juniper NetworksÂ®

Last updated: Mar 13, 2019

Optimization and Visualization enables cloud operators and users to monitor, visualize, and control how infrastructure resources are utilized by workloads and plan adequate capacity to ensure application performance. It also provides visibility into operational performance of infrastructure resources.

For more information about RPCO Optimization and Visualization, see the AppFormix Overview in the AppFormix User Guide.

This guide is intended to assist customers in setting up and using the capabilities of RPCO Optimization and Visualization.
