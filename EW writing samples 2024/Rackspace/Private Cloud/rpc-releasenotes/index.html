Release Notes#

Rackspace Private Cloud Powered By OpenStack

Last updated: Mar 13, 2019

This document describes new features and known and resolved issues in Rackspace Private Cloud Powered By OpenStack (RPCO), release 14.0 (r14-dev).

RPCO 14.0 uses open source Ansible playbooks to deploy an OpenStack Newton private cloud. The RPCO repository is available at https://github.com/rcbops/rpc-openstack. For more information about the OpenStack-Ansible Newton release, see the Release Notes or OpenStack Configuration Reference.
