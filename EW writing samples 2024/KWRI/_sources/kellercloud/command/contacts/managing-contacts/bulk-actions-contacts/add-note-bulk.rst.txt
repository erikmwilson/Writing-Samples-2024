Add a Note to Multiple Contacts
===============================

You can add a note to multiple contacts using the Bulk Action feature.

To add a note to multiple contacts, complete the following steps:

.. note::

   You can also add notes to individual contacts. For more information, see :ref:`add-note-contacts-command`.

.. include:: contacts.rst

#. On the left toolbar, click **Contacts** |contacts-icon|.

   .. |contacts-icon| image:: ../../../../../images/contacts-icon.png
      :height: 20

#. Select the check boxes next to the contacts you want to add a note to. To select all contacts, select the check box in the header row.

   .. note::

      You can filter your contact list using Filters to target certain criteria (for example, **Primary Address**).

#. Click the **Select Bulk Action** drop-down menu and select **Add Note**.

#. On the **Add New Note** window, enter the **Note Title** and **Note Description**.

#. Click **Save**.
