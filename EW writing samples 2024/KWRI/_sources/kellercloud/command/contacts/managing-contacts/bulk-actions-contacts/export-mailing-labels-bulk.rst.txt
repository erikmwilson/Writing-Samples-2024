Export Mailing Labels for Multiple Contacts
===========================================

To export mailing labels for multiple contacts, complete the following steps:

.. include:: contacts.rst

#. On the left toolbar, click **Contacts** |contacts-icon|.

   .. |contacts-icon| image:: ../../../../../images/contacts-icon.png
      :height: 20

#. Select the check boxes next to the contacts for which you want to export mailing labels. To select all contacts, select the check box in the header row.

   .. note::

      You can filter your contact list using Filters to target certain criteria (for example, **Primary Address**).

#. Click the **Select Bulk Action** drop-down menu and select **Export Mailing Labels**.

#. The following confirmation message appears, confirming that the export was successful: "Your export was successful, please check your notifications".

#. Click the notifications |notification-icon| icon, and within the export label notification, click the link to download the exported labels.

   .. |notification-icon| image:: ../../../../../images/system-notifications-icon.png
      :height: 20
