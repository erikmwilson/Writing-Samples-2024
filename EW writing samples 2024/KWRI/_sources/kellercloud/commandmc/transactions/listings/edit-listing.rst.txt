Edit Listing
============

To edit a listing, complete the following steps:

.. include:: transactions-mc-login.rst

#. Click the **Listings** tab at the top of the page.

#. Search for the correct listing. For more information, see :ref:`search-listings-commandmc`.

#. Click the ellipses icon |menu-options-icon| (on the far right) for that listing and select **Edit**.

   .. |menu-options-icon| image:: ../../../../images/menu-options-icon.png
      :height: 20

#. On the **Edit Listing** window, edit the listing information and click **Save**.
