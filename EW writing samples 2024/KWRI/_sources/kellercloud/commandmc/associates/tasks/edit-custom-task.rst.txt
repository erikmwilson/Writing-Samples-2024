Edit a Custom Task
==================

To edit a custom task, complete the following steps:

.. include:: tasks-mc-login.rst

#. Click **Other Tasks** from the drop-down. A list displays of all the custom tasks.

#. Click **To Do**. Only tasks in this status can be edited.

#. Locate the appropriate task, click the More Options menu (ellipses |menu-options-icon| icon) and select **Edit**.

   .. |menu-options-icon| image:: ../../../../images/menu-options-icon.png
      :height: 20

#. On the **Edit Task** window, update the information as necessary.

   .. note::

      Because each task is tied to a specific associate, you cannot change the associate name. Instead, delete the task and create a new task for the new associate.

#. Click **Edit Task**.
