Add Tags to Multiple Recruits
=============================

You can add tags to multiple recruits using the Bulk Action feature.

To add tags to multiple recruits, complete the following steps:

.. include:: recruits-mc-login.rst

#. Click the **Recruit Management** tab.

#. Select the check boxes next to the recruits to which you want to add tags. To select all recruits, select the check box in the header row.

   .. note::

      You can filter your recruits list using Filters to target certain criteria (for example, **Pipeline Status**).

#. Click **Add A Tag** under the recruits list.

#. On the **Add Tags** window, select the correct tags. To create a custom tag, enter the tag name and click **Create Custom Tag**.

#. Click **Save**.
