Add a SmartPlan to Multiple Recruits
====================================

You can add a SmartPlan to multiple recruits using the Bulk Action feature. You can also add SmartPlans to individual recruits. For more information, see :ref:`add-smartplan-recruits-commandmc`.

To add a SmartPlan to multiple recruits, complete the following steps:

.. include:: recruits-mc-login.rst

#. Click the **Recruit Management** tab.

#. Select the check boxes next to the recruits to which you want to add a SmartPlan. To select all recruits, select the check box in the header row.

   .. note::

      You can filter your recruits list using Filters to target certain criteria (for example, **Pipeline Status**).

#. Click **Add to SmartPlan** under the recruits list.

#. On the **Add Recruits to SmartPlan** window, find the correct SmartPlan and click **Select**.

#. Select when you want the SmartPlan to start:

   * **Start all now** - The first step of the SmartPlan begins immediately, as soon as you confirm.

   * **Start all on the following date** - Use the calendar widget to choose a date on which to start the SmartPlan.

#. Click **Confirm**.

#. On the **Add to SmartPlan** pop-up, review the details, then click **Confirm**.
